# 软件项目管理作业

## 项目名称

密接和次密接人员自动检索

## 项目背景

自3月2日吉林市疫情出现疫情以来，疫情逐渐变得严重，我们从最开始的封校到封寝，我们大家都亲身经历了这次疫情，我们深刻的体会到疫情之下没有人能够袖手旁观，于是我们团队便想充分利用我们的专业，为抗疫献出绵薄之力，所以启动了该项目。
